Gain insights into your SQL data flows

Real time monitoring

DF Tracker tracks sql files in your GitHub repositories, analyses them and provides you with an interactive data landscape that updates as soon as the state of any repository changes.

Simple usage

1. Sign In with GitHub

2. Tell DFTracker which repositories to track

3. Enjoy the view :)

Respecting your privacy

DF Tracker works by performing static code analysis. We do not need connections to your SQL servers. Non of your scripts is stored in our databases.
